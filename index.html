مشخصات VPS تو این اطلاعات از فروشنده VPS داده شده

IP و پورت SSH سرورت رو وارد کن — بقیه مراحل خودکار تنظیم می‌شن.

آدرس IPv4 که فروشنده VPS بهت داده — معمولاً یه عدد ۴ قسمتی مثل ۱۲.۳۴.۵۶.۷۸
بیشتر سرورها پورت ۲۲ دارن — اگه فروشنده عدد دیگه‌ای داده همون رو بزن.
۱
اتصال به سرور با Termius اپ SSH رایگان برای موبایل

Termius یه اپ رایگانه که بهت اجازه می‌ده از طریق موبایل دستور روی سرور اجرا کنی.

📲 دانلود Termius از Play Store
راهنمای اتصال (قدم به قدم):
۱. Termius رو باز کن
۲. روی دکمه + (گوشه پایین راست) بزن
۳. New Host رو انتخاب کن
۴. در فیلد Hostname: IP سرورت رو بنویس
۵. در فیلد Port: پورت SSH رو بنویس
۶. در فیلد Username: root بنویس
۷. در فیلد Password: رمز سرورت رو بنویس
۸. Save کن، سپس روی هاست تَپ کن
اگه پیام Are you sure you want to continue connecting? اومد، Yes بزن — این یه هشدار امنیتی نرماله.
۲
به‌روزرسانی سرور حدود ۲ تا ۵ دقیقه طول می‌کشه

این دستور رو در ترمینال Termius پیست کن. منتظر بمون تا کامل بشه.

دستور به‌روزرسانی + نصب ابزارها
apt update -y && apt upgrade -y && apt install -y curl wget socat sqlite3 ca-certificates ufw
اگه سؤال پرسید:
• [Y/n] نوشت؟ → حرف Y بزن و Enter
• صفحه آبی یا بنفش باز شد؟ → فقط Enter بزن (یا کلیدهای جهت‌نما رو بزن تا OK بیاد)
• هیچ کاری نکرد و فقط نوشت Done → نرمالیه، بریم مرحله بعد
۳

پنل حتی بدون SSL گواهی‌دار هم کار می‌کنه. کانفیگ نهایی به TLS پنل وابسته نیست — Netlify این کار رو می‌کنه.

۴
ساخت خودکار اتصال UUID و PATH تصادفی ساخته می‌شن

این اسکریپت خودکار یه اتصال VLESS+XHTTP روی پورت ۸۴۴۳ می‌سازه، پورت رو باز می‌کنه و مقادیر رو نمایش می‌ده.

اسکریپت ساخت inbound — همه رو یکجا کپی و پیست کن
در حال آماده‌سازی...
بعد از اجرا، ترمینال چیزی شبیه این نشون می‌ده:
================================
  inbound created successfully
================================
  UUID: d1e825b6-9d29-...
  PATH: relay-vmt3u6
================================
مقادیر رو از ترمینال اینجا وارد کن
کل عدد ۳۶ کاراکتری روبروی UUID: رو کپی کن.
فقط همون چیزی که روبروی PATH: نوشته — بدون / اول یا آخر.
۵
دانلود فایل پروژه Netlify IP سرورت خودکار داخل فایل جاسازی می‌شه
ابتدا IP سرور رو در مرحله ۰ وارد کن

با کلیک روی این دکمه، یه فایل ZIP ساخته می‌شه که آدرس سرور تو از قبل داخلش تنظیم شده. هیچ تنظیم اضافه‌ای در Netlify لازم نیست.

آدرس سرور جاسازی‌شده: https://...:8443
فایل دانلود نشد؟ مطمئن شو که:
• IP سرور رو در مرحله ۰ وارد کردی
• مرورگرت دانلود رو مسدود نکرده (بالای مرورگر نگاه کن)
۶
آپلود در Netlify ثبت‌نام رایگان — هیچ تنظیمی لازم نیست

فایل ZIP که دانلود کردی رو به Netlify آپلود کن.

🌐 باز کردن Netlify Drop
راهنمای آپلود (قدم به قدم):
۱. لینک بالا رو باز کن (اگه اکانت نداری، ثبت‌نام رایگان کن)
۲. صفحه‌ای با نوشته Drag and drop می‌بینی
۳. روی browse files to upload بزن
۴. فایل netlify-relay.zip رو انتخاب کن
۵. صبر کن تا وضعیت به Published ✓ تغییر کنه
بعد از deploy موفق، یه آدرس مثل amazing-site-abc123.netlify.app در بالای داشبورد نشون داده می‌شه. این آدرس رو کپی کن — در مرحله بعد لازمه.
آپلود رد شد یا خطا داد؟
• اگه Edge Functions not supported گفت: در Netlify باید Free plan داشته باشی
• اگه Build failed گفت: فایل ZIP رو دوباره در مرحله ۵ دانلود و آپلود کن

فقط آدرس بدون https:// — مثلاً: my-relay-abc.netlify.app

ایمپورت کانفیگ در v2rayNG:
۱. کانفیگ بالا رو کپی کن
۲. v2rayNG رو باز کن
۳. دکمه + (بالا راست) → Import config from Clipboard
۴. کانفیگ ایمپورت می‌شه — روش تَپ کن تا انتخاب بشه
۵. دکمه پلِی (▶) پایین صفحه رو بزن
۶. اگه اتصال برقرار شد، آیکون V2 در نوار بالا سبز می‌شه
